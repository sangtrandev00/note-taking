## Data type lưu trữ key

+ Object
	+ chỉ là chuỗi (string)
+ Map
	+ Có thể là bất kỳ kiểu dữ liệu nào
## Thứ tự lưu trữ key

- Object: 
	- Thứ tự lưu trữ không theo thứ tự
- Map
	- Thứ tự lưu trữ theo thứ tự insert (insertion order)
